All tools are in the left side navigation bar. Some tools are grouped by category, which is accessible by toggling its sub-menu in the sidebar.

Data Storage

Utilitron uses your browser's local storage to store and retrieve any settings and data you may create. This means that this data is not stored on any server, and therefore only present on the browser you use to access this application with. If you use another browser (i.e. on another device), the settings and data you saved originally (on the other browser) will not be present.
